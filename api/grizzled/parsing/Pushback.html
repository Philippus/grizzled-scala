  17.  def pushbackMany(items: List[T]): Unit

    Push a list of items back onto the stream.

    Push a list of items back onto the stream. The items are pushed back in reverse order, so the items in the list should be in the order they were retrieved from the stream. For example:

    val stream = new SafeIterator[Char]("foobar") with Pushback[Char]
val list = List(stream.next.get, stream.next.get)

// At this point, the list contains ('f', 'o'), and the stream
// contains "obar".

stream.pushback(list) // Stream now contains "foobar"
    items

    the items to push back.
