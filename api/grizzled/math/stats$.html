object stats extends AnyRef

Miscellaneous statistics-related functions.

Note: You must import scala.math.Numeric (or just Numeric._) for these functions to work. For example:

import Numeric._
import grizzled.math.stats._

val l = List[Double]( ... )
println(median(l))
