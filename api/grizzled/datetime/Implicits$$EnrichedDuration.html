    Return a better-formatted result than toString.

    Return a better-formatted result than toString. For instance, given this Duration:

    Duration(13123, "milliseconds")

    the standard toString method will return "13123 milliseconds". By contrast, humanize will return "13 seconds, 123 milliseconds".

    Similarly:

    val d = Duration(172801003, "milliseconds")

d.toString     // 172801003 milliseconds
d.humanize // 2 days, 1 second, 3 milliseconds
