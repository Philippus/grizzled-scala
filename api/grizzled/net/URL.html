URL

Companion object URL

final case class URL(protocol: String, host: Option[String], port: Option[Int], path: Option[String], query: Option[String] = None, userInfo: Option[String] = None, fragment: Option[String] = None) extends Product with Serializable

Convenient Scala case-class wrapper for a java.net.URL. This class doesn't include all the capabilities. For example, it lacks the equivalent of getContent(), as that's better handled through other means.
