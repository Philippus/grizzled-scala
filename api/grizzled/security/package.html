  1. object MessageDigest

    Convenience methods for generating crypto-hash (i.e., message digest) strings (e.g., MD5, SHA256, etc.) from byte values.

    Convenience methods for generating crypto-hash (i.e., message digest) strings (e.g., MD5, SHA256, etc.) from byte values. Sample use:

    // Generate an MD5 hash-string of a string

val hash = MessageDigest("md5").digestString("foo")

// Generate an SHA256 hash-string of a file's contents.

val hash = MessageDigest("sha256").digestString(new File("/tmp/foo"))
