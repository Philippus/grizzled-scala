  8. def complete (token: String, context: List[CompletionToken], line: String): List[String]

    Get all completions for a token.

    Get all completions for a token. The context argument bears some explaining. It's designed to allow the completer to locate the cursor (and the nearest token) via Scala pattern matching, and it consists of a stream of abstract tokens:

    • LineToken is a token parsed from the line, in an object that's similar to Some: LineToken.value returns the token's string.
    • Cursor indicates the location of the cursor
    • Delim indicates the presence of a token delimiter (typically white space)

    The input line is broken into these tokens, which can then be matched. For example, consider the following input lines, with the cursor where the caret is:

    Input Tokens
    `^` `Cursor`
    `cm^` `LineToken("cm") Cursor`
    `cmd ^` `LineToken("cmd") Delim Cursor`
    `cmd arg1^` `LineToken("cmd") Delim Token("arg1") Cursor`
    `cmd arg1^` `LineToken("cmd") Delim Token("arg1") Delim Cursor`
    `cmd^ arg1` `LineToken("cmd") Cursor Delim Token("arg1")`
    `cm^d arg1` `LineToken("cmd") Cursor Delim Token("arg1")`

    token

    the token being completed

    line

    the current unparsed input line, which includes the token

    returns

    a list of completions, or Nil if there are no matches
